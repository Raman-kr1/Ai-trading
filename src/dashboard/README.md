# 📁 Dashboard (`/src/dashboard`)

Web-based trading dashboard with real-time stats and monitoring.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Single-page dashboard with dark theme, live stats, trade history, AI decisions |

## Features

- **Portfolio Stats**: Total PnL, win rate, profit factor, AI accuracy
- **Trade History**: Recent trades with side, entry, status, PnL
- **AI Decisions**: Complete audit log of Claude's decisions
- **Queue Control**: View BullMQ job status
- **Auto-refresh**: Updates every 30 seconds

## Access

```
http://localhost:3000/dashboard
```
